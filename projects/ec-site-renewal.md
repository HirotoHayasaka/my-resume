月間100万PVのECサイトをNext.js + TypeScriptでフルリニューアル。
レガシーなPHPアプリケーションからの移行プロジェクト。

## 技術スタック

- Next.js 13 (App Router) / TypeScript
- Tailwind CSS / Radix UI
- PostgreSQL / Prisma
- Vercel / GitHub Actions

- コンポーネント設計のガイドラインを策定
- A/Bテスト基盤の構築によりCVR 15%向上
