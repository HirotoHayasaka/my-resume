## 概要

レガシーな決済基盤をGoベースのマイクロサービスアーキテクチャにリプレイスするプロジェクト。
年間数十億円規模のトランザクション処理を支える基盤システムの刷新を担当。

## 技術スタック

- Go 1.21 / gRPC / Protocol Buffers
- PostgreSQL 15 / Redis
- AWS ECS Fargate / ALB / CloudWatch
- GitHub Actions CI/CD

## 担当・成果

- アーキテクチャ設計とチームリードを担当
- レスポンスタイムを平均200ms→50msに改善（75%削減）
- ダウンタイムゼロでの段階的移行を実現
- テストカバレッジ85%以上を維持
